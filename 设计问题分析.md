### 2.2 开源组件选型

| 功能模块 | 选型 | 优势 |
|----------|------|------|
| 资产管理 | NetBox | 机柜图、IPAM、REST API |
| 监控告警 | Zabbix | SNMP模板、告警规则、API |
| 运维工单 | GLPI | ITIL流程、API |
| 日志审计 | Graylog | 集中日志、搜索 |
| 统一认证 | LDAP/AD | 企业目录集成 |
| 备份 | BorgBackup | 增量去重 |

## 三、设计原则

### 3.1 小规模适配

- 避免过度设计
- 选用轻量级方案
- Docker Compose 部署

### 3.2 稳定优先

- 成熟开源组件
- 完整 API 接口
- 丰富社区支持

### 3.3 易于维护

- 标准化部署
- 统一认证
- 集中日志
