## 十二、日志审计

### 12.1 日志采集方案

#### 12.1.1 工具选型：Graylog

| 特性 | Graylog | 说明 |
|------|---------|------|
| **日志收集** | ✅ | GELF、Syslog |
| **日志存储** | ✅ | Elasticsearch |
| **日志搜索** | ✅ | 全文搜索 |
| **仪表盘** | ✅ | 可视化展示 |
| **告警** | ✅ | 基于日志内容告警 |

#### 12.1.2 日志采集架构

```
┌─────────────────────────────────────────────────────────────┐
│                   Graylog 日志架构                           │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│   [应用日志] ──► [Filebeat] ──► [Graylog] ──► [存储]   │
│   [系统日志] ──► [Syslog]  ──► [Graylog] ──► [存储]   │
│   [安全日志] ──► [Winbeat] ──► [Graylog] ──► [存储]   │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

### 12.2 审计日志内容

| 日志类型 | 来源 | 内容 |
|----------|------|------|
| **登录日志** | 各系统 | 用户登录/登出 |
| **操作日志** | 自建API | 数据增删改查 |
| **变更日志** | NetBox | 资产变更 |
| **告警日志** | Zabbix | 告警触发/确认 |
| **工单日志** | GLPI | 工单状态变更 |
