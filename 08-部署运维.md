# 网络运维监控平台 - 部署运维

## 八、部署架构

### 8.1 部署方案选型

#### 8.1.1 推荐方案：Docker Compose

对于小规模系统（3机房、1000+VM），推荐使用 **Docker Compose** 部署，简单易维护。

| 部署方式 | 适用场景 | 复杂度 |
|----------|----------|--------|
| **Docker Compose** | 小规模、简单 | 低 |
| Kubernetes | 大规模、弹性 | 高 |

### 8.3 Docker Compose 配置

```yaml
# docker-compose.yml

version: '3.8'

services:
  # 自建 API 服务
  api:
    image: network-monitor/api:latest
    ports:
      - "8080:8080"
    environment:
      - DB_HOST=postgres
      - REDIS_HOST=redis
      - NETBOX_URL=http://netbox:8000
      - ZABBIX_URL=http://zabbix:8080
      - GLPI_URL=http://glpi:80
    depends_on:
      - postgres
      - redis
      - netbox
      - zabbix
      - glpi

  # 前端
  web:
    image: network-monitor/web:latest
    ports:
      - "3000:80"

  # NetBox (资产管理)
  netbox:
    image: netboxcommunity/netbox:latest
    ports:
      - "8000:8000"
    volumes:
      - netbox-media:/opt/netbox/netbox/media

  # Zabbix (监控)
  zabbix:
    image: zabbix/zabbix-server-pgsql:latest
    ports:
      - "8080:8080"
    environment:
      - DB_SERVER_HOST=postgres
      - POSTGRES_USER=zabbix
      - POSTGRES_PASSWORD=zabbix

  # GLPI (工单)
  glpi:
    image: linuxserver/glpi:latest
    ports:
      - "80:80"
    volumes:
      - glpi-data:/config

  # 数据库
  postgres:
    image: timescale/timescaledb:latest-pg16
    volumes:
      - pgdata:/var/lib/postgresql/data

  # 缓存
  redis:
    image: redis:7-alpine

volumes:
  netbox-media:
  glpi-data:
  pgdata:
```

### 8.4 组件配置

#### 8.4.1 NetBox 配置

```bash
# 环境变量
NETBOX_SECRET_KEY=your-secret-key
ALLOWED_HOSTS=netbox.company.com
DATABASE_URL=postgres://user:pass@postgres:5432/netbox
REDIS_URL=redis://redis:6379/0
```

#### 8.4.2 Zabbix 配置

```bash
# 环境变量
DB_SERVER_HOST=postgres
POSTGRES_USER=zabbix
POSTGRES_PASSWORD=zabbix
ZBX_CACHESIZE=512M
ZBX_HISTORYCACHESIZE=512M
```

### 8.5 备份方案

#### 8.5.1 备份工具选型

| 工具 | 说明 | 推荐场景 |
|------|------|----------|
| **BorgBackup** | 增量备份、去重压缩 | 文件备份 |
| **pg_dump** | PostgreSQL逻辑备份 | 数据库备份 |
| **Velero** | Kubernetes备份 | K8s环境 |

#### 8.5.2 备份脚本示例

```bash
#!/bin/bash
# backup.sh

# 数据库备份
pg_dump -h postgres -U zabbix zabbix > /backup/zabbix_$(date +%Y%m%d).sql

# NetBox 备份
docker exec netbox /opt/netbox/backup.sh

# 增量备份
borg create /backup::$(hostname)-$(date +%Y%m%d) /data

# 清理7天前备份
find /backup -type f -mtime +7 -delete
```
