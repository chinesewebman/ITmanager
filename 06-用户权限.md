# 网络运维监控平台 - 用户权限管理

## 六、用户权限管理详细设计

### 6.1 认证方案：LDAP/AD 集成

#### 6.1.1 认证架构

| 组件 | 说明 |
|------|------|
| **LDAP/AD** | 企业目录服务 (OpenLDAP / Windows AD) |
| **LDAP认证模块** | 系统内置 LDAP 认证 |
| **SSO (可选)** | CAS / OAuth2 / SAML |

#### 6.1.2 支持的目录服务

| 服务 | 支持 | 说明 |
|------|------|------|
| **Windows AD** | ✅ 原生 | 企业常用 |
| **OpenLDAP** | ✅ 原生 | 开源方案 |
| **389 Directory** | ✅ 配置 | Fedora目录服务 |
| **Apple Open Directory** | ✅ 配置 | macOS服务器 |

### 6.2 权限模型

#### 6.2.1 RBAC + LDAP 组映射


| **API用户** | IT-API | 通过API访问资源 |

### 6.3 各系统认证配置

#### 6.3.1 NetBox LDAP 配置

```python
# netbox/netbox/configuration.py

AUTHENTICATION_BACKENDS = [
    'netbox.authentication.LDAPBackend',
    'django.contrib.auth.backends.ModelBackend',
]

# LDAP 服务器配置
LDAP_SERVER_URI = "ldap://ldap.company.com"
LDAP_BIND_DN = "cn=admin,dc=company,dc=com"
LDAP_BIND_PASSWORD = "your-password"

# 用户搜索
LDAP_USER_SEARCH = LDAPSearchUnion(
    LDAPSearch(
        "ou=users,dc=company,dc=com",
        ldap.SCOPE_SUBTREE,
        "(uid=%(user)s)"
    ),
    LDAPSearch(
        "ou=admins,dc=company,dc=com",
        ldap.SCOPE_SUBTREE,
        "(sAMAccountName=%(user)s)"
    ),
)

# 组映射
LDAP_GROUP_SEARCH = LDAPSearch(
    "ou=groups,dc=company,dc=com",
    ldap.SCOPE_SUBTREE,
    "(objectClass=groupOfNames)"
)

# NetBox 组同步
LDAP_GROUP_PARAMS = {
    "netbox_admin": {
        "name": "NetBox Admin",
        "is_superuser": True,
    },
    "netbox_operator": {
        "name": "NetBox Operator",
        "is_staff": True,
    },
}
```

#### 6.3.2 Zabbix LDAP 配置

```
# Zabbix Web 界面配置
# Administration > Authentication > LDAP

设置:
- LDAP host: ldap://ldap.company.com
- Port: 389
- Base DN: dc=company,dc=com
- Search attribute: uid
- Bind DN: cn=admin,dc=company,dc=com
- Bind password: ********

用户映射:
- Zabbix user: uid={username},ou=users,dc=company,dc=com
```

```
服务器配置:

同步选项:
```

### 6.4 多因素认证 (MFA)

#### 6.4.1 支持的MFA方式

| 方式 | 系统支持 | 说明 |
|------|----------|------|
| **TOTP** | NetBox / Zabbix | Google Authenticator |
| **短信** | GLPI | 需要短信网关 |
| **邮件** | 各系统 | 验证码 |
| **硬件Key** | NetBox | YubiKey 等 |

### 6.5 会话管理

#### 6.5.1 会话配置

| 参数 | 默认值 | 说明 |
|------|--------|------|
| Token有效期 | 24小时 | 可配置 |
| 刷新Token | 7天 | 长时间访问 |
| 会话超时 | 30分钟 | 无操作超时 |
| 密码策略 | 复杂度要求 | 8位+大小写+数字 |

#### 6.5.2 安全控制

- 登录失败锁定: 5次失败后锁定15分钟
- 密码过期: 90天 (可配置)
- 登录IP限制: 支持白名单
